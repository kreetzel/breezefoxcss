# breezefoxcss
Firefox userChrome.css made to match KDE's Breeze theming

# Prereq:
- Enable toolkit.legacyUserProfileCustomizations.stylesheets , widget.gtk.rounded-bottom-corners.enabled , svg.context-properties.content.enabled in about:config
- Set widget.use-xdg-desktop-portal.file-picker to 1 in about:config
